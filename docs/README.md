## Enfoque de la Documentación

Esta documentación está enfocada en:
- **Reglas de negocio**: Cómo funciona la lógica del negocio
- **Lógica condicional**: Cuándo y cómo se aplican las reglas
- **Información almacenada**: Qué datos se guardan de cada entidad
- **Interfaz de usuario**: Dónde están los botones, qué hacen, qué colores tienen
- **Flujos de trabajo**: Cómo se realizan las operaciones principales
- **Procesos automáticos**: Qué se ejecuta automáticamente y cuándo

**NO incluye**:
- Detalles técnicos de implementación
- Ejemplos de código
- Configuración técnica
- Detalles de arquitectura

## Uso para Agentes IA

Esta documentación puede ser consultada por agentes IA para:
- Entender cómo funciona la plataforma
- Saber qué información está disponible
- Conocer las reglas de negocio
- Identificar dónde están los elementos de la interfaz
- Comprender los flujos de trabajo
- Entender los procesos automáticos
